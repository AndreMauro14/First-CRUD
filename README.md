# Primeiro CRUD - Projeto Pessoal

Este é meu primeiro projeto CRUD (Create, Read, Update, Delete) desenvolvido durante o curso de Programação Orientada a Objetos (POO) do instrutor Nélio Alves.

Estou muito feliz com a minha evolução na programação e com os conceitos que estou aprendendo para construir aplicações Java usando boas práticas de desenvolvimento.

Espero continuar melhorando e explorando novos desafios na área de desenvolvimento de software!

***

## Sobre o Projeto

- Projeto feito em Java, utilizando JDBC para conexão com banco de dados.
- Implementa operações básicas de CRUD para a entidade Departamento.
- Prática importante para fixar conceitos de POO, banco de dados e integração Java-SQL.

## Minha motivação

Esse projeto marca um momento importante na minha trajetória como programador. Estou cada vez mais motivado a evoluir e encarar novos desafios na área de tecnologia.
